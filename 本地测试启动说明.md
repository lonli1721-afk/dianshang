# 顽皮素材工具本地测试版

这个目录是从线上最新 Git 提交 `8e94c24` 导出的干净源码，用来本地测试，不会影响线上服务器。

## 启动方式

在 PowerShell 里进入本目录后运行：

```powershell
Set-ExecutionPolicy -Scope Process Bypass
.\start-local-test.ps1
```

默认端口：

- 前端页面：http://127.0.0.1:6180
- 后端接口：http://127.0.0.1:6181
- 本地测试数据：本目录下的 `.local-data`

## 注意

- 第一次启动会自动安装 Python 和前端依赖，时间会久一点。
- 本地版需要单独在设置页填写测试用 API Key。
- 本地测试产生的账号、任务、图片、视频都存在 `.local-data`，不会写入线上服务器。
- 测试 OK 后，再把代码提交并走线上部署流程。
